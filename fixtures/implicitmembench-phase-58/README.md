Phase 58 Targeted ImplicitMemBench Subset
=========================================

This fixture directory mirrors a mechanism-focused adapted subset of the upstream ImplicitMemBench benchmark for Phase 58 raw enactment compiler work.

Source benchmark:
- https://github.com/qinchonghanzuibang/ImplicitMemBench
- License: CC BY 4.0

Scope of this mirror:
- conflict-to-inhibition from failure/correction experience
- text-response deterministic repair for forbidden surfaces and safe replacements
- exact prefix/suffix and procedural text contracts
- structured first-action exactness for command, wrapper, argument-order, and slot-preserving APIs

The subset contains 50 one-instance blocking case files so each smoke assertion is bound to the instance-level expected behavior. It is internal research/runtime evidence only; it is not a public release benchmark claim and does not replace full-300 follow-up research runs.
