Phase 54 Targeted ImplicitMemBench Subset
=========================================

This fixture directory mirrors a small adapted subset of the upstream
ImplicitMemBench benchmark for targeted Phase 54 raw internalization work.

Source benchmark:
- https://github.com/qinchonghanzuibang/ImplicitMemBench
- License: CC BY 4.0

Scope of this mirror:
- raw exemplar-first carryover for same-scope behavior transfer
- interference-aware suppression for semantically similar but incompatible cues
- exact host-action carryover for command, tool-call, and argument-order tasks
- bounded procedural transfer without collapsing single exemplars into prose rules
- raw text-response carryover without visible memory-note phrasing

This is an internal eval subset, not a vendored copy of the full upstream
repository.
