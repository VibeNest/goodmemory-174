Phase 31: Native Host Outcome and Correction Closure
====================================================

Phase Goal
----------
Strengthen the accepted Phase 30 Codex host path without reopening it.

This phase upgrades live behavioral evidence from:
- native host first-action capture plus fixture-scored synthetic outcome

to:
- native `codex exec --json` event stream
- native `HostBehavioralTraceEvent`
- host-derived executable outcomes
- native targeted correction lineage

The scope is internal-only evidence hardening. It does not widen the public API, public config, or README-level default behavior.
