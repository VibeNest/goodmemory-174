Write tests first
-----------------

- conflicting or weak procedural suggestions do not promote prematurely
- pattern scope stays bounded by user/workspace/agent applicability

Implementation steps
--------------------

Acceptance

Implemented Notes
-----------------
- Accepted `procedural_pattern` proposals now compile into active `validated_pattern` feedback records through an idempotent compiler pass.

- Source guidance is superseded instead of silently deleted, and compiled patterns retain inspectable rationale/evidence lineage.
