Write tests first
-----------------
- public type-surface tests prove event input is available only on the intended adapter packages
- invalid or partial event payloads fail closed
- installed-package consumer smoke can import the new adapter-level types/helpers without `src/*`

Acceptance
----------
- event input is package-public and Bun-installable
- adapter-level event input carries `scope`, host identity, and per-turn provenance
- every public agent event carries stable dedupe/lineage keys:
  - `eventId`
  - `runId` or `attemptId`
  - `turnId`
  - `sequence`
  - `occurredAt`
  - optional `parentEventId`
- event input can be omitted with zero behavior change to the accepted pre-Phase-31 path
- event input does not require the user to instantiate internal reviewer/proposal/runtime objects directly
- public naming keeps agent-input events distinct from the existing recall/remember telemetry event family
