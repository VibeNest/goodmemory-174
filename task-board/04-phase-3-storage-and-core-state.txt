Phase 3: Storage and Core State
===============================

Phase Goal
----------
Implement storage contracts and the minimum persistence backplane required by the public API and tests.


Dependencies
------------
- Phase 2 complete


Primary Deliverables
--------------------
- Storage interfaces
- In-memory adapter
- SQLite adapter
- Postgres adapter contract and skeleton
- session state persistence contract


Definition of Done
------------------
- In-memory adapter passes integration tests
- SQLite adapter passes the same contract tests
- Postgres adapter contract is defined and partially scaffolded
