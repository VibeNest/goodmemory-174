Phase 60.1: Overall Score Contract
==================================

Goal
----
Create a stable report contract for official-comparable full-300 scoring.

Tasks
-----
- [x] Add a typed score object for:
  - `blockingScore`
  - `primingScore`
  - `full300OverallScore`
  - `overallComparableToOfficial`
  - denominator and official-comparability details
- [x] Keep the existing blocking metric unchanged:
  `passedBlockingCases / totalBlockingCases`.
- [x] Define how each priming pair contributes to the full-300 denominator.
- [x] Store both raw priming judge scores and normalized overall-credit values.
- [x] Add tests that fail when a report has blocking numbers but no official
  comparable overall section.
- [x] Add tests that prove historical Phase 49 reports are read as legacy
  blocking-only evidence rather than silently treated as full-overall evidence.

Acceptance
----------
- A Phase 60 report can state, without ambiguity, whether the run is:
  - blocking-only
  - full-300 but not official-comparable
  - official-comparable full-300
- The formula is visible in JSON, not only in prose documentation.
