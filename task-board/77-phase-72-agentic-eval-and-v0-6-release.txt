Phase 72: Agentic Eval and v0.6 Release
======================================

Status
------

[ACTIVE]

Goal
----

Validate generalized memory on HaluMem and a MemGym coding slice, retain
LongMINT as smoke-only, refresh external reproducibility evidence, and prepare
the `v0.6.0` release.

Gate
----

- New non-judge LLM calls use `gpt-5.6-terra` through `ai.gurkiai.com`, persist
  the exact model role and gateway identity, and use a different model for any
  LLM-judge protocol.
- Current benchmark thresholds and zero-failure requirements pass; HaluMem
  beats the simple-vector baseline and MemGym coding beats no-memory by 5
  points; typecheck, tests, coverage, Node package smoke, Playwright, and pack
  gates pass with unpacked size at or below 4 MB.

Release Decision
----------------

Do not bump the package version, create a tag, or publish `v0.6.0` while the
current full-answer score gates remain open. Historical declarations stay
versioned evidence; `benchmarks.currentClaims` remains empty.
