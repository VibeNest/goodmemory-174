Phase 66: v0.3 Release Readiness and Public Surface Hardening
============================================================

Status
------

[ACTIVE] Phase 66 is the current lane. After the Phase 62-65 sequential
benchmark-hardening work produced enough internal evidence (LongMemEval accepted;
BEAM measured checkpoint 0.695; MemoryAgentBench accepted internal AR/CR
zero-failure live closure; LoCoMo banked retrieval boundary), the next priority
is NOT more benchmarks. It is proving the v0.3 public package surface is
installable, importable, explainable, and rollback-safe for real users.

This phase is docs + release-tooling + examples only. It opens no new eval/code
work, does not reopen LoCoMo performance work (semantic candidate-generation is
only local plumbing until external-root evidence exists; see board 70), does not
touch the Phase 63 BEAM answer-pack lane (owned by the BEAM workstream), and
adds no new public benchmark claim.


Driver
------

GoodMemory has scattered internal benchmark evidence but the public surface had
drift: stale phase wording across docs, undocumented new recall knobs (multiHop,
createLocalEmbeddingAdapter), and no single gate that proves the package ships
and imports cleanly. A user installing `goodmemory@0.3.5` must be able to install,
import, run, and understand it without being misled by README/docs.


Scope / Non-Goals
-----------------

Do:
- Sync benchmark claim boundaries across README, Current-Status, Plan, and board.
- Document the new public recall knobs with anti-misuse cautions.
- Ship a reproducible release-readiness gate (build/pack/import/CLI/consistency).
- Smoke the real example entry points.

Do NOT:
- Promote LoCoMo semantic-candidate research into defaults or public claims
  before broader sharded coverage plus adversarial/noise/open-domain follow-up.
- Run the local hashed-embedding hybrid (a lexical proxy; would pollute results).
- Touch the Phase 63 BEAM answer-pack workstream's code.
- Put MAB / LoCoMo internal numbers into the public README benchmark table.
- Add any new public benchmark claim.
