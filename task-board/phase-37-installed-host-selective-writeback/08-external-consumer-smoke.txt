Phase 37.8: External Consumer Smoke
===================================

Goal
----
Verify Phase 37 from outside the GoodMemory repository so adoption risk is tested on the installed package path.


Fixture Shape
-------------
Create a temporary consumer fixture:

```text
.tmp-goodmemory-phase37-consumer/
  package.json
  .goodmemory/codex.json
  scenario/
    session-1.json
    session-2.json
```


Smoke Chain
-----------
Run the packed-package flow:

```bash
npm pack
cd .tmp-goodmemory-phase37-consumer
npm install ../goodmemory-0.1.1.tgz
./node_modules/.bin/goodmemory install codex --user-id phase37 --no-interactive
./node_modules/.bin/goodmemory enable codex --writeback selective
cat scenario/session-1.json | ./node_modules/.bin/goodmemory codex writeback --json
cat scenario/session-2.json | ./node_modules/.bin/goodmemory codex hook user-prompt-submit
```

Adjust the tarball path/version to the actual package name produced by `npm pack`.
