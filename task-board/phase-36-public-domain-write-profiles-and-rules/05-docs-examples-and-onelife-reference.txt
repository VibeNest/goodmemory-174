OneLife Reference Scenario
--------------------------
The example should demonstrate:

- long-term goal capture
- current focus or life project capture
- habit and routine preference capture
- relationship context capture
- coaching style preference capture
- explicit correction and supersession
- assistant suggestions not becoming durable unless confirmed


Acceptance

- docs never claim OneLife is built in
- examples work with public imports only
- examples do not rely on `testing.extractor`
